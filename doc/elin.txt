CUSTOMIZING                                                   *elin-customizing*

                                                           *g:elin_auto_connect*
g:elin_server_auto_connect
  If `v:true`, automatically connect to Elin server.
  Default value is `v:true`.

                                                            *g:elin_server_port*
g:elin_server_port
  Elin server port number.
  If `v:null`, automatically assing a empty port number.
  Default value is `v:null`.

                                            *g:elin_enable_default_key_mappings*
g:elin_enable_default_key_mappings
  If `v:true`, enable default key mappings.

                                             *g:elin_default_key_mapping_leader*
g:elin_default_key_mapping_leader
  Default value is `'<Leader>'`.

                                                 *g:elin_enable_omni_completion*
g:elin_enable_omni_completion
  If `v:true`, enable omni completion.
  Default value is `v:true`.

                                        *g:elin#internal#nrepl#reload_before_fn*
g:elin#internal#nrepl#reload_before_fn
  A qualified function name to evaluate before |ElinReload|, |ElinReloadAll|.
  The function should be zero-arity.
  E.g. >
  let g:elin#internal#nrepl#reload_before_fn = 'user/stop'
<
  Default value is `''`. (No function to evaluate)

                                         *g:elin#internal#nrepl#reload_after_fn*
g:elin#internal#nrepl#reload_after_fn
  A qualified function name to evaluate after |ElinReload|, |ElinReloadAll|.
  The function should be zero-arity.
  E.g. >
  let g:elin#internal#nrepl#reload_after_fn = 'user/start'
<
  Default value is `''`. (No function to evaluate)
